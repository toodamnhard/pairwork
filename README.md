
# 开发环境及算法结构

#### 搭建环境

- 编程语言：java
- 工具：intellij idea

#### 算法结构

src目录下

- pojo

  - 定义学生信息的`Student`类

- data
  - 随机生成绩点的`CreateJidian`类
  - 随机生成到勤信息的`CreateChuqin`类
  - 数据集生成器`CreateShuru`类

- algorithm
  - 获取数据集的`GetShuru`类
  - 点名并输出E的`Choudian`类

#### 运行代码

- 首先运行`CreateShuru`类，生成数据集
- 最后运行`Choudian`类，输出E
